# Iris Flower Classification using Decision Tree

## Overview
This project builds a Decision Tree classification model to predict the species of iris flowers based on their physical measurements. The classic Iris dataset is used to demonstrate supervised machine learning concepts.

## Dataset
- **Source:** Iris.csv (UCI Machine Learning Repository)
- **Features:** Sepal length, sepal width, petal length, petal width
- **Target:** Iris species (Setosa, Versicolor, Virginica)
- **Records:** 150 samples

## Tools & Technologies
- Python
- Pandas & NumPy
- Scikit-learn (Decision Tree Classifier)
- Matplotlib / Seaborn (Visualisation)
- Jupyter Notebook

## What I Did
- Loaded and explored the Iris dataset
- Performed exploratory data analysis (EDA)
- Built a Decision Tree classification model using Scikit-learn
- Evaluated model performance using accuracy score and confusion matrix
- Visualised the decision tree structure

## Key Results
- Successfully classified iris species using petal and sepal measurements
- Decision Tree model achieved high accuracy on test data
- Visualised feature importance showing petal measurements are strongest predictors
